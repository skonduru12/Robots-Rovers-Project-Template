# Claude Rules
## Document types:
1) Only return standard rich text Mark Down files. Resulting rich text mark down will be cut and pasted into Google docs, slides, sheets.
2) Never return docx or pdf
3) When creating documents, site any sources, with links, at the end of the document.
## Inclusive language in reporting, tables, and diagrams:
1) Use Controller/Target rather than Master/Slave terminology describing pins and protochols. The industry has been moving away from Master/Slave terminology for a while now, with I²C itself officially adopting Controller/Target in 2021.
